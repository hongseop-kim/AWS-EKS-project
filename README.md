# HighFive_team

# 프로젝트 설명
여러가지 OTT플랫폼의 컨텐츠를 우리가 구축한 웹 사이트에서 간편하게 확인.
3-TIER 아키텍처를 통하여 각 계층을 분리하고 서로 영향을 미치치 않아 독립적인 확장이 원활하게 가능.
Amazon EKS(Elastic Kubernetes Service)를 통해 쿠버네티스 클러스터를 구축 후 오토스케일링, 로드밸런싱을 통한 부하 분산이 되도록 구현.
